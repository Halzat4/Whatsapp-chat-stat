# WhatsApp Chat Intelligence & Topic Modeler

Продвинутый инструмент для глубокого анализа и предобработки логов WhatsApp. Проект выполняет не только базовую статистику, но и нормализует сленг, фильтрует части речи и находит скрытые темы в переписке с помощью машинного обучения.

## Основные возможности

* **Глубокая нормализация (Normalization 2.0):** * Исправление опечаток и сокращений через кастомный `NORM_MAP` (например, "пон" -> "понял", "мб" -> "maybe").
    * Обработка смешанных языков (RU/EN) с использованием `pymorphy3` и `SnowballStemmer`.
    * Интеллектуальная обработка смеха (все вариации "ахаха" сводятся к единому токену).
* **Лингвистическая фильтрация:** * Удаление стоп-слов, мусора и ненормативной лексики.
    * Фильтрация по частям речи (POS-tagging) для повышения качества анализа тем.
* **Аналитика и Визуализация:**
    * Построение графиков активности по часам.
    * Рейтинг самых активных участников.
    * Топ частотных слов.
* **Тематическое моделирование (LDA):**
    * Автоматическое определение основных тем обсуждения с помощью модели `Gensim LDA`.

## Структура проекта

* `Ml22.py` — точка входа. Запуск анализа и построение LDA-модели.
* `preprocessing.py` — "движок" предобработки. Содержит логику очистки, маппинга сленга и фильтрации контента.
* `analytics.py` — модуль агрегации данных, работы с Pandas и визуализации (Matplotlib).
* `perfect_dictionary.txt` — справочный словарь чистых токенов.

## Установка и запуск

1.  **Требования:** Python 3.10+
2.  **Установка зависимостей:**
    ```bash
    pip install pandas matplotlib nltk pymorphy3 gensim
    ```
3.  **Подготовка данных:**
    Поместите файл с экспортом чата в `data/halz_tima.txt`.
4.  **Запуск:**
    ```bash
    python Ml22.py
    ```

## Пример работы

Скрипт автоматически:
1. Выведет в консоль статистику по сообщениям.
2. Откроет окно с тремя графиками (активность, топ авторов, частота слов).
3. Обучит LDA-модель на 3 основные темы и выведет ключевые слова для каждой темы.

## Настройка
Вы можете редактировать `TextConfig` в файле `preprocessing.py`, чтобы добавить свои специфичные стоп-слова или правила замены сленга в `NORM_MAP`.
