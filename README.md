# WhatsApp Chat Preprocessor & Analyzer

Скрипт на Python для обработки экспортированных логов чата WhatsApp. Выполняет очистку данных, лемматизацию (для русского и английского языков) и базовый статистический анализ.

## Функционал
* **Очистка системных сообщений:** Удаление уведомлений о сквозном шифровании и сервисных строк.
* **Нормализация имен:** Автоматическая замена заданных имен контактов.
* **Лемматизация:** * Для русского языка используется `pymorphy3` (приведение к начальной форме).
  * Для английского языка используется `SnowballStemmer`.
* **Фильтрация:** Удаление стоп-слов и пунктуации.
* **Статистика:** Подсчет количества сообщений по отправителям и вывод топ-10 самых частотных слов.

## Требования
Для работы требуются следующие библиотеки:
* `nltk`
* `pymorphy3`
